Shadows

All shadows are provided as custom properties. To access them the var() CSS function can be called with the shadow's name:

.some-example {
  box-shadow: var(--theme-shadow-1);
}
--theme-shadow-1
--theme-shadow-2
--theme-shadow-3
--theme-shadow-4
--theme-inset-shadow-1
